# Supported Chains

{% hint style="warning" %}
**Pre-release** — The full list of supported chains and assets will be published ahead of launch.
{% endhint %}

Intents Connect is designed to support any combination of source and destination chains, including EVM and non-EVM networks.

All chains supported by Near Intents will be supported by Intents Connect:

* Ethereum
* Bitcoin
* NEAR
* Solana
* Binance Smart Chain
* Base
* Arbitrum
* Cardano
* Sui
* TON
* Polygon
* Optimism
* Zcash
* Tron
* Avalanche
* Bera
* XRP
* Gnosis
* Dogecoin
* Stellar
* Litecoin

That said, at launch, coverage will include a subset of these chains. We will then roll out more support as the product develops.

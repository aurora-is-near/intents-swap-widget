# Supported Chains

{% hint style="warning" %}
**Pre-release** — The full list of supported chains and assets will be published ahead of launch.
{% endhint %}

Intents Connect is designed to support any combination of source and destination chains, including EVM and non-EVM networks.

All chains supported by Near Intents will be supported by Intents Connect:

{% tabs %}
{% tab title="EVM Chains" %}
* Ethereum
* Binance Smart Chain
* Base
* Arbitrum
* Polygon
* Optimism
* Bera
* Gnosis
* Avalanche
* Monad
{% endtab %}

{% tab title="Bitcoin & Forks" %}
* Bitcoin (coming soon)
* Zcash (coming soon)
* Dogecoin (coming soon)
* Litecoin (coming soon)
{% endtab %}

{% tab title="Layer 1 Chains" %}
* Solana
* NEAR (coming soon)
* Cardano (coming soon)
* Sui (coming soon)
* TON (coming soon)
* Tron (coming soon)
* XRP (coming soon)
* Stellar (coming soon)
{% endtab %}
{% endtabs %}

That said, at launch, coverage will include a subset of these chains. We will then roll out more support as the product develops.
